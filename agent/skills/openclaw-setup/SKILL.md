# OpenClaw Setup

Help install OpenClaw, choose models (cloud and/or local Ollama), run the gateway,
and connect messaging channels the user wants.

- Prefer current upstream install docs for flags that churn  
- Store tokens in env or the tool’s secret mechanism — never commit them  
- Verify with a health/status command after start  
- Document how to stop/restart the gateway on this machine
