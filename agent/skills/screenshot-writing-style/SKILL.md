# Screenshot Writing Style

While handling screenshots for the main task, if text is clearly by the user
(identity from style profile handles), feed durable patterns into
`{MEMORY_ROOT}/user_writing_style.md`. Treat OCR as medium confidence; do not
overwrite high-confidence typed patterns with one blurry frame.
